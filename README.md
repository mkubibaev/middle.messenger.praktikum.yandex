Веб мессенджер - учебный проект курса Яндекс.Практикум "Мидл фронтенд-разработчик"
---
* Сборка проекта 
    ```shell script
        npm run build
    ```

* Локальный запуск express.js 
    ```shell script
        npm run start
    ```

* Локальный запуск parcel.js 
    ```shell script
        npm run serve
    ```
